# Лазер ТК — Отслеживание заказов

Страница отслеживания статуса заказа для [lasertk.ru](https://lasertk.ru)

## Как работает

1. Клиент вводит номер заказа и телефон
2. Система проверяет данные в Supabase
3. Показывает статус с прогресс-баром (5 этапов)

## Стек

- Vanilla HTML/CSS/JS
- Supabase (база данных)
- GitHub Pages (хостинг)
